# README

Application for kilometrique fees

To install the Rails Application


```
git clone git@github.com:selimchiki/popkilometrique.git
```
or
```
git clone https://github.com/selimchiki/popkilometrique.git
```

```
cd popkilometrique
bundle install
rails db:create
rails db:migrate
rails db:seed
rails server
```
